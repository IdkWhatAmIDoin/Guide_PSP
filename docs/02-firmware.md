# 02 — updating firmware

[← back to main guide](../README.md)

> [!NOTE]
> skip this page if you're already on **6.60 or 6.61**. check via `settings → system settings → system information`.

---

## option a — online update (easiest, but probably dead)

`settings → network update`

sony's update servers are likely offline. if it works, great. if not, use option b.

---

## option b — update via memory stick

1. download the official firmware from sony:
   - [psp firmware 6.61](https://www.playstation.com/en-us/support/hardware/psp/update/)
2. on your memory stick, create this path: `/PSP/GAME/UPDATE/`
3. rename the downloaded file to `EBOOT.PBP` and drop it in that folder
4. slot the memory stick into your psp and go to `game → memory stick`
5. run the update

[this wikihow guide](https://www.wikihow.com/Upgrade-Your-PSP-Firmware) has screenshots if you get stuck.

> [!WARNING]
> keep your psp plugged in or at 75%+ battery during the update. don't power it off mid-flash or you'll brick it.
