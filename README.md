## universal requirements

- firmware **6.60 or 6.61** — check via `settings → system settings → system information`
- a **memory stick pro duo** (or pro-hg duo / pro-hg duo hx)
- a **memory stick reader** (a microsd adapter works fine)
- **[ARK-4 latest release](https://github.com/PSP-Archive/ARK-4/releases/latest)** — download from github

> [!NOTE]
> this guide uses ARK-4, the actively maintained successor to PRO and ME cfw. if you see older guides recommending PRO-C2, ARK-4 is the better choice now.
