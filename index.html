guide

a guide to running custom firmware on your psp · all models

some info is sourced from wololo.net and other sites. originally written for psp-3000, but the steps are the same across all models.
requirements what you need
  • psp on firmware 6.60 or 6.61 — check via settings › system settings › system information
  • memory stick pro duo, pro-hg duo, or pro-hg duo hx
  • pro cfw files for 6.60 or 6.61
  • a memory stick reader (a microsd adapter works great)
step 01 updating firmware
⌦ skip if you're on 6.60 or 6.61 already

online update is the easiest but the servers are probably dead. your best bet is updating via memory stick — this wikihow guide covers it.

step 02 copying files to memory stick

plug your memory stick into a reader and extract the cfw zip. copy the PSP folder to the root of your memory stick — replace all if prompted. paste the seplugins folder in too.

step 03 installing cfw on the psp

eject the memory stick, slot it into your psp, head to the games menu and run PRO-C Update. cfw is now installed.

heads up: cfw resets on power off. run Fast Recovery every time you restart — or see below for the permanent option.
optional permanent cfw (psp 1000 / 2000 only)

download pspident, copy it to the memory stick and run it. if you have a psp 1000 or 2000 and the motherboard isn't 88 v3, you can install permanent cfw — no more fast recovery.

you're done 🎉

there's an ISO folder on your memory stick now (create it if missing). drop psp game isos in there and they'll show up in your games menu. enjoy.
