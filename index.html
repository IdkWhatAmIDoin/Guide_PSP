a guide to running custom firmware on your psp · all models

some info is sourced from wololo.net and other sites. originally written for psp-3000, but the steps are the same across all models.
requirements what you need
  • psp on firmware 6.60 or 6.61 — check via settings › system settings › system information
  • memory stick pro duo, pro-hg duo, or pro-hg duo hx
  • ARK-4 latest release — download the zip from github
  • a memory stick reader (a microsd adapter works great)
step 01 updating firmware
⌦ skip if you're on 6.60 or 6.61 already

online update is the easiest but the servers are probably dead. your best bet is updating via memory stick — this wikihow guide covers it.

keep your psp plugged in or at 75%+ battery during the update. don't power it off mid-flash or you'll brick it.
step 02 copying files to memory stick

plug your memory stick into a reader and extract the ARK-4 zip. copy ARK_01234 to /PSP/SAVEDATA/ and copy ARK_Loader to /PSP/GAME/.

step 03 installing cfw on the psp

eject the memory stick, slot it into your psp, head to the game menu and run ARK Loader. the psp will soft restart — you're now running ark-4.

heads up: cfw resets on full power off. relaunch ARK Loader after each cold boot — or install permanent cfw below (works on all models including psp-3000).
optional permanent cfw (psp 1000 / 2000 only)

with ark-4 running temporarily, copy ARK_cIPL to /PSP/GAME/ and run ARK cIPL Flasher. works on all models including psp-3000 — no motherboard check needed.

you're done 🎉

there's an ISO folder on your memory stick now (create it if missing). drop psp game isos in there and they'll show up in your game menu. enjoy.
